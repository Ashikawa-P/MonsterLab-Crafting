Simple html that helps at crafting in MonsterLab (PS2, DS)
Pictures are all extracted from the monsterlab wiki!
Selecting a primary ingedient will open a chart of all possible Monster parts that can be build with it
-Machine in the first row
-biological in the second
-alchemical in the third
selecting a part opens a collumn showing:
-at the top the enchantment this part gets
-the secondary ingedients needed to get the enchantment
